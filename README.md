<h1>Life Expectancy in 193 Countries 2000-2015</h1>

<h2>Description</h2>
Project consists of a visualization using Looker Studio (Google Data Studio) about life expectancy in 193 countries 2000 - 2015.
<br />

<h2>Languages and Utilities Used</h2>

- <b>Google Spreadsheet</b> 
- <b>Looker Studio</b>

<h2>Environments Used </h2>

- <b>Windows 11</b>
